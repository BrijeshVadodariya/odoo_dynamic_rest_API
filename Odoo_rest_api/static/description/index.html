Installation Guide

  1. Download the module from our repository or the Odoo App Store
  2. Place it in your Odoo 18 addons directory: /path/to/odoo18/addons/
  3. Update your app list: Go to Apps ‚Üí Update Apps List
  4. Install the REST API Module from the Apps menu
  5. Configure user permissions: Settings ‚Üí Users & Companies ‚Üí Users
    • Enable "API Access" for relevant users
    • Set appropriate access rights for API users
  6. Restart your Odoo 18 service
  7. Verify installation by accessing: https://your-odoo-domain.com/api/status

Configuration Options:

  • API Rate Limiting - Configure in Settings ‚Üí REST API Configuration
  • CORS Settings - Define allowed origins for cross-domain requests
  • Default Response Format - Choose between JSON or XML output
  • Logging Level - Set verbosity of API request logging
  • Token Expiration - Set JWT token validity period (default 24 hours)

API Documentation

All API requests are made to the base URL:

https://your-odoo-domain.com/jsonrpc

Authentication:

To authenticate, send a POST request to /api/login with credentials:

{
  "jsonrpc": "2.0",
  "method": "call",
  "params": {
    "db": "your_db_name",
    "login": "api_user@example.com",
    "password": "secure_password"
  }
}

Successful authentication returns a JWT access token valid for 24 hours by default.

Available Endpoints:

  • POST /api/<model>/search - Search records

    Parameters: domain (array), limit (int), offset (int), order_by (string), fields (array)

  • GET /api/<model>/read - Read record by ID

    Parameters: id (int), fields (array)

  • POST /api/<model>/create - Create new record

    Parameters: values (object)

  • PUT /api/<model>/update - Update existing record

    Parameters: id (int), values (object)

  • DELETE /api/<model>/delete - Delete record

    Parameters: id (int)

  • POST /api/<model>/call_method - Call model method

    Parameters: method (string), args (array), kwargs (object)

Error Handling:

The API returns standard HTTP status codes with detailed error messages:

  • 400 - Bad Request (invalid parameters)
  • 401 - Unauthorized (invalid credentials)
  • 403 - Forbidden (insufficient permissions)
  • 404 - Not Found (resource doesn't exist)
  • 429 - Too Many Requests (rate limit exceeded)
  • 500 - Internal Server Error

Postman Usage

We provide a ready-to-use Postman collection specifically configured for Odoo 18's REST API. Download it from our documentation site or GitHub repository.

Setup Instructions:

  • Import the collection into Postman
  • Set environment variables:
    • base_url - Your Odoo 18 instance URL
    • db - Your database name
    • username - API user email
    • password - API user password
  • Run the "Authentication" request to get your access token
  • Token will automatically be set for subsequent requests

Example Request:

Search for companies with more than 10 employees:

{
  "jsonrpc": "2.0",
  "method": "call",
  "params": {
    "domain": [
        ["is_company", "=", true],
        ["employee_count", ">", 10]
    ],
    "limit": 5,
    "order_by": "name asc",
    "fields": ["id", "name", "email", "phone"]
  }
}
